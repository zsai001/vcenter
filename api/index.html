API 参考 ​

VCenter 提供 RESTful API 用于与系统交互。

基础信息 ​

属性值
Base URLhttp://your-server:8090/api/v1
协议HTTP/HTTPS
数据格式JSON
认证方式Bearer Token (JWT)

认证 ​

获取 Token ​

http
POST /api/v1/auth/login
Content-Type: application/json

{
  "username": "admin",
  "password": "your-password"
}

响应：

json
{
  "success": true,
  "data": {
    "access_token": "eyJhbGciOiJIUzI1NiIs...",
    "refresh_token": "eyJhbGciOiJIUzI1NiIs...",
    "expires_at": "2024-12-01T12:00:00Z"
  }
}

使用 Token ​

在请求头中添加：

http
Authorization: Bearer eyJhbGciOiJIUzI1NiIs...

响应格式 ​

成功响应 ​

json
{
  "success": true,
  "data": { ... },
  "message": "操作成功"
}

错误响应 ​

json
{
  "success": false,
  "error": {
    "code": "INVALID_INPUT",
    "message": "输入参数无效"
  }
}

分页响应 ​

json
{
  "success": true,
  "data": {
    "items": [...],
    "pagination": {
      "page": 1,
      "page_size": 20,
      "total": 100,
      "total_pages": 5
    }
  }
}

HTTP 状态码 ​

状态码说明
200请求成功
201创建成功
400请求参数错误
401未认证
403无权限
404资源不存在
500服务器错误

系统 API ​

健康检查 ​

检查服务是否正常运行。

http
GET /health

响应：

json
{
  "success": true,
  "data": {
    "status": "healthy",
    "service": "vcenter",
    "version": "1.0.0"
  }
}

节点管理 API ​

获取节点列表 ​

page_sizeint每页数量，默认 20
statusstring状态筛选 (online/offline)
group_idstring分组 ID

        "cpu_cores": 8,
        "memory_total": 16106127360,
        "last_seen_at": "2024-12-01T10:00:00Z",

错误码说明 ​

错误码说明
INVALID_INPUT输入参数无效
NOT_FOUND资源不存在
UNAUTHORIZED未认证或 Token 无效
FORBIDDEN无操作权限
NODE_NOT_CONNECTED节点未连接
CLUSTER_NOT_CONNECTED集群未连接
INTERNAL_ERROR内部服务错误

SDK 与工具 ​

cURL 示例 ​

bash
# 获取 Token
TOKEN=$(curl -s -X POST http://localhost:8090/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"admin123"}' \
  | jq -r '.data.access_token')

# 获取节点列表
curl -s http://localhost:8090/api/v1/nodes \
  -H "Authorization: Bearer $TOKEN" | jq

Go 客户端 ​

go
package main

import (
    "fmt"
    "net/http"
)

func main() {
    client := &http.Client{}
    req, _ := http.NewRequest("GET", "http://localhost:8090/api/v1/nodes", nil)
    req.Header.Set("Authorization", "Bearer YOUR_TOKEN")
    
    resp, _ := client.Do(req)
    defer resp.Body.Close()
    // ...
}
