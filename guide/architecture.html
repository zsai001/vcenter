系统架构 ​

VCenter 采用分层架构设计，确保系统的可维护性和可扩展性。

产品定位 ​

VCenter 是 VStaff 产品套件的核心组件，专为企业级集群管理设计：

产品定位说明
VPanel单节点管理轻量级服务器管理面板
VCenter企业集群管理多节点与 K8s 集群统一管理
VCloud云服务中心用户认证、License、官网

核心模块 ​

1. 节点管理 ​

管理通过 Agent 连接的 VPanel 节点：

  • 节点注册 - 自动发现和注册
  • 状态监控 - 实时状态和指标
  • 远程命令 - 远程执行命令
  • 分组管理 - 节点分组和标签

2. Kubernetes 管理 ​

多 K8s 集群统一管理：

  • 集群管理 - 添加和管理多个集群
  • 工作负载 - Deployment, StatefulSet, DaemonSet
  • 网络资源 - Service, Ingress
  • 配置存储 - ConfigMap, Secret, PV/PVC
  • 日志终端 - Pod 日志和终端

3. 监控告警 ​

  • 资源使用监控
  • 自定义告警规则
  • 多渠道通知

4. 用户权限 ​

基于角色的访问控制 (RBAC)：

角色权限
Admin全部权限
Operator操作权限
Viewer只读权限

数据流 ​

用户请求 → Gin Router → Middleware → Handler → Service → Database
                           ↓
                      认证/授权检查
                           ↓
                        请求日志
                           ↓
                      审计日志记录

安全设计 ​

  • 认证: JWT Token
  • 传输: TLS 加密
  • 密码: bcrypt 哈希
  • 审计: 操作日志
  • 限流: 请求频率限制
